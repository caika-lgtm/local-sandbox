# Decision Record Template

Append completed records to `decisions.md`.

```markdown
### DXXX: Short decision title

- Status: Proposed | Accepted | Rejected | Superseded
- Date: YYYY-MM-DD
- Owner: TBD
- Related milestone: MNN
- Related RFC section: N

#### Context

What forced this decision? Include implementation evidence, failed attempts, or validation results.

#### Decision

State the decision in one or two precise sentences.

#### Consequences

- Positive consequence.
- Negative or tradeoff consequence.
- Required follow-up.

#### Alternatives considered

- Alternative A: reason rejected.
- Alternative B: reason rejected.
```
